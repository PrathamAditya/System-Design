Created a bare-minimum terminal chat application to observe how bidirectional communication works between a client and server.

Used **Socket.IO** to establish the two-way event-driven connection and **Werkzeug** as the WSGI server to host the Socket.IO application.

The goal of this small experiment was to understand the basic mechanics of real-time communication — connection handshake, message emission, and event handling — in the simplest possible setup.
